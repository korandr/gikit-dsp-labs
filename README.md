# Лабораторные работы ЦОАВС

Репозиторий содержит лабораторные работы по курсу «Цифровая обработка аудиовизуальных сигналов».
Вся практическая часть оригинальных методических указаний, изначально ориентированная на
проприетарную среду MATLAB 7, портирована на открытый стек Python 3.

## Литература

Цифровая обработка аудиовизуальных сигналов : метод. указания по выполнению лабораторных работ для
студентов дневного отделения по специальности 210312 «Аудиовизуальная техника». — Санкт-Петербург : 
СПбГУКиТ, 2009. — 26 с.

## Системные требования и окружение

- ОС: протестировано на MacOS/Linux, но c большой веротностью будет работать и на Windows.
- Версия Python: 3.14.
- Ключевые зависимости: `numpy`, `scipy`, `matplotlib`.

## Установка (MacOS/Linux)

```bash
# Клонирование репозитория
git clone https://github.com/korandr/gikit-dsp-labs.git
cd gikit-dsp-labs

# Настройка виртуального окружения
python3.14 -m venv .venv
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

Запуск лабораторных работ:
```bash
python src/labN.py
```

## Лабораторная работа №1. Аналогово-цифровое преобразование

Работа посвящена моделированию процесса преобразования непрерывного сигнала в цифровой (АЦП).

В практической части исследуется, как частота дискретизации влияет на форму сигнала (проверка
теоремы Котельникова), а также тестируются 4 метода округления (`round`, `ceil`, `floor`, `trunc`)
и оценивается их ошибка (шум) квантования.

- [Задания 1 и 2](src/lab1_12.py)
- [Задания 3, 4 и 5](src/lab1_34.py)

### Соответствие функций MATLAB и Python

| MATLAB           | Python                          | Комментарий                                                                                              |
| :--------------- | :------------------------------ | :------------------------------------------------------------------------------------------------------- |
| `n = 0:Td:99`    | `n = np.arange(0, 100, Td)`     | Создание массива отсчётов. Правая граница в Python не включается.                                        |
| `plot(n, y)`     | `plt.plot(n, y)`                | Построить непрерывную линию по точкам.                                                                   |
| `stem(n, y)`     | `plt.stem(n, y)`                | Отобразить отсчёты в виде вертикальных линий с точками на концах.                                        |
| `stairs(n, y)`   | `plt.step(n, y, where='mid')`   | Отобразить ступенчатый график (как на выходе АЦП).                                                       |
| `hist(ERR, x)`   | `plt.hist(ERR, x)`              | Отобразить гистограмму                                                                                   |
| `round(y)`       | `np.round(y)`                   | Математическое округление к ближайшему целому.                                                           |
| `ceil(y)`        | `np.ceil(y)`                    | Округление в сторону плюс бесконечности.                                                                 |
| `floor(y)`       | `np.floor(y)`                   | Округление в сторону минус бесконечности.                                                                |
| `fix(y)`         | `np.trunc(y)`                   | Округление в сторону нуля.                                                                               |
| `mean(err)`      | `np.mean(err)`                  | Среднее арифметическое (постоянная составляющая ошибки).                                                 |
| `std(err)`       | `np.std(err, ddof=1)`           | Cтандартное отклонение (мощность шума квантования). `ddof=1` нужен для несмещенной оценки, как в MATLAB. |
